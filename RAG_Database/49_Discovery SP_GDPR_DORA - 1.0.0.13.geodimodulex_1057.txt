Bulk tagging and classification can be performed.

Country-specific semantic recognizers extend the capabilities of GEODI Discovery and Classification. This package is for Spain and should be installed after GEODI Discovery.

The package includes the following recognizers: CIF Recognizer, DNI Recognizer, NIE Recognizer, NIF Recognizer, SSN Recognizer, and Spain Passport Number Recognizer.

The version of this package is 1.0.0.13.
